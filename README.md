# Pacoff — Keeping Offline Systems Up-to-Date

This is a simple-to-use tool with Pacman-like syntax to download packages for a system without Internet — that can be an air-gapped system or just a desktop on a connection with a temporary outage.


## Installation

Just copy the `pacoff` file to a place under `path`:
```none
# cp --update=none-fail ./pacoff /usr/local/bin/pacoff
```
and you're done!
